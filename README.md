## Notes
* Game was tested on:
  * iPhone SE
  * Samsung Galaxy S6
  * Samsung Galaxy Tab S2
  * Samsung Galaxy Tab S
  * Huawei P10 Lite
  * Windows 10 (Laptop with Touchscreen)
* The game relies on touch gestures, therefore it cannot be played in the Unity editor (unless you have a touchscreen).
